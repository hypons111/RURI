## 2024-02-10
其實 searchBar 同 table 係應該寫喺一個組件，但我又想試吓子組件同子組仔之間互傳資料，所以夾硬寫埋一齊。
原理：父元件宣告一個 **響應式變數** 同一個 **用嚟改呢個變數嘅方法** 傳俾 **A子組件** ，再將呢個變數傳俾 **B子組件**，就可以 **A** 傳 **B**。
